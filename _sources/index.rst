Modèle relationnel
==================

.. rubric:: Contenus étudiés:

- Modèle relationnel : relation, attribut, domaine, clef primaire, clef étrangère, schéma relationnel
- Base de données relationnelle

.. rubric:: Capacités attendues

- Identifier les concepts définissant le modèle relationnel. Ces concepts permettent d’exprimer les contraintes d’intégrité (domaine, relation et référence)
- Savoir distinguer la structure d’une base de données de son contenu. Repérer des anomalies dans le schéma d’une base de données.
